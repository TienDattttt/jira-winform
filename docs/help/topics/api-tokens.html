API token

Tạo và thu hồi token cá nhân để dùng với local REST API của ứng dụng.

Thực hiện

  1. Mở phần hồ sơ cá nhân hoặc khu vực quản lý token tương ứng.
  2. Chọn Create API Token.
  3. Nhập tên token, chọn thời hạn hết hạn và tick các scope cần thiết như ReadIssues, WriteIssues, ReadProjects hoặc ManageProjects.
  4. Xác nhận tạo token và sao lưu giá trị token ngay khi dialog hiển thị.
  5. Khi token không còn cần dùng, thu hồi token để vô hiệu hóa ngay.
Giá trị token chỉ nên được lưu trong nơi quản lý bí mật an toàn. Không gửi token qua email hoặc chụp ảnh màn hình dùng chung.
