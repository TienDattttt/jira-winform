Quản lý người dùng hệ thống

Tạo user, cập nhật thông tin, đổi mật khẩu và điều chỉnh trạng thái tài khoản ở cấp hệ thống.

Điều kiện trước khi thao tác

  • Tài khoản đang đăng nhập có quyền Admin ở cấp hệ thống.

Thực hiện

  1. Mở menu Users ở sidebar.
  2. Dùng màn hình quản lý người dùng để tạo user mới hoặc mở user hiện có.
  3. Cập nhật họ tên, email, trạng thái active hoặc system role khi cần.
  4. Nếu người dùng quên mật khẩu, dùng dialog đặt lại mật khẩu để cấp mật khẩu mới.

Lưu ý

  • Việc tạo user hệ thống không tự động thêm user đó vào project nào.
  • Sau khi tạo user, cần thêm membership ở từng project nếu người đó cần nhìn thấy project trong ứng dụng.
