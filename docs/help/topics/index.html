Jira Clone Desktop

Tài liệu trợ giúp cho bản ứng dụng WinForms.

Tài liệu này hướng dẫn sử dụng ứng dụng theo góc nhìn người dùng cuối và quản trị viên. Nội dung tập trung vào thao tác thực tế trong môi trường vận hành, không bao gồm checklist test nội bộ.

Dùng mục lục bên trái để mở nhanh từng chủ đề hoặc nhấn các liên kết bên dưới.

Nội dung chính

  • Người dùng: đăng nhập, điều hướng, xử lý issue, làm việc với board, backlog, roadmap, báo cáo và xuất dữ liệu.
  • Quản trị viên: quản trị dự án, quản trị hệ thống, workflow, permission, tích hợp và vận hành.
  • Troubleshooting: cách xử lý các lỗi thường gặp khi đăng nhập, truy cập dữ liệu, xuất báo cáo và mở file trợ giúp.

Phạm vi áp dụng

Nhóm người dùng Mục tiêu chính Màn hình sử dụng nhiều
Viewer / Developer Theo dõi tiến độ, cập nhật issue, cộng tác trong dự án Tổng quan, Bảng, Backlog, Lộ trình, Issue, Báo cáo
Project Manager / Admin dự án Điều hành sprint, quản lý thành viên, workflow, cấu hình dự án Sprint, Backlog, Cài đặt dự án, Báo cáo
System Admin Quản trị tài khoản, cấu hình hệ thống và vận hành tổng thể Users, Settings, API token, tích hợp, webhook

Vai trò và quyền

Viewer: chỉ xem dữ liệu trong dự án mà mình là thành viên.

Developer: xử lý issue, cập nhật trạng thái, comment, attachment, watcher.

Project Manager: quản lý sprint, board, thành viên, workflow, labels, components, versions, webhooks và tích hợp của dự án.

Admin: có quyền quản trị hệ thống và/hoặc quyền cao nhất trong project tùy theo vai trò được gán.

Cách mở tài liệu trợ giúp

  1. Mở ứng dụng và chọn menu Trợ giúp trong màn hình chính.
  2. Ứng dụng sẽ mở file JiraClone.chm từ thư mục cài đặt.
  3. Nếu file trợ giúp không mở được, xem mục Không mở được file trợ giúp.
