Tài liệu trợ giúp Jira Clone Desktop

Tài liệu vận hành dành cho người dùng cuối, quản trị viên dự án và bộ phận hỗ trợ.

Tài liệu này mô tả đúng các chức năng hiện có của ứng dụng WinForms: đăng nhập, làm việc với project, issue, board, sprint, roadmap, báo cáo, webhook, tích hợp và API token. Cấu trúc tài liệu đi theo kiểu help file production với topic tổng quan và topic thao tác cụ thể.

Đây là tài liệu phục vụ sử dụng và hỗ trợ vận hành. Tài liệu này không thay thế checklist test, tài liệu mã nguồn hoặc tài liệu API dành cho lập trình viên.

Đường dẫn nhanh theo vai trò

Vai trò Mục tiêu chính Topic nên đọc trước
Viewer / Developer Theo dõi và cập nhật công việc hằng ngày trong project Getting Started, Main Interface, Create a New Issue, Update an Issue, Work with the Scrum Board hoặc Work with the Kanban Board
Project Manager / Project Admin Lập kế hoạch sprint, cấu hình project và theo dõi tiến độ bàn giao Manage Sprints, Workflow and Board Settings, View Charts and Sprint Reports, Members and Permissions
System Admin / Support Quản lý người dùng, kết nối và các sự cố vận hành Install the Desktop App, Configure Connection and Storage, System User Management, Troubleshooting

Cấu trúc tài liệu

  • Introduction: tổng quan sản phẩm, mô hình vận hành và đối tượng sử dụng.
  • Installation and Setup: cài desktop client, cấu hình kết nối SQL Server và thư mục lưu attachment.
  • Getting Started: đăng nhập, chọn project và làm quen với shell.
  • Projects and Issues: tạo, cập nhật, tìm kiếm và phối hợp xử lý issue.
  • Boards and Sprints: thao tác trên Scrum hoặc Kanban board và quản lý vòng đời sprint.
  • Roadmap and Reports: xem roadmap, biểu đồ, sprint report và các tùy chọn export.
  • Administration: quản lý thành viên, quyền, workflow, webhook, tích hợp và API token.
  • Troubleshooting: lỗi thường gặp về đăng nhập, database, attachment, webhook, export và file trợ giúp.

Cách dùng tài liệu này

  • Các topic thao tác mô tả điều kiện trước khi thực hiện, các bước thực hiện và kết quả mong đợi.
  • Mỗi nhóm tính năng chính có một topic cha và nhiều topic con cho các tác vụ thường gặp.
  • Dùng tab Index để nhảy nhanh tới các từ khóa như Sprint, Epic, Workflow, Webhook hoặc API token.
