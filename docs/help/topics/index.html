Jira Clone Desktop Help

Tài liệu trợ giúp cho bản ứng dụng desktop dùng trong môi trường vận hành nội bộ.

Cấu trúc tài liệu này đi theo dạng help file thực tế cho phần mềm desktop: giới thiệu, cài đặt, bắt đầu nhanh, giao diện, tác vụ chính, quản trị và troubleshooting. Nội dung chỉ mô tả các chức năng hiện có trong hệ thống.

Đây là tài liệu phục vụ sử dụng và vận hành. Tài liệu này không thay thế checklist test, tài liệu mã nguồn hoặc tài liệu API.

Main sections

  • Introduction: tổng quan sản phẩm, vai trò sử dụng và yêu cầu môi trường.
  • Installation and Setup: cài đặt client, cấu hình kết nối và kiểm tra sau cài đặt.
  • Getting Started: đăng nhập, chọn project và luồng bắt đầu làm việc.
  • User Interface: bố cục màn hình chính, điều hướng, search, notification và help.
  • Projects and Issues: project, issue, comment, attachment và watcher.
  • Board and Sprint: board, backlog, sprint và nguyên tắc thao tác theo workflow.
  • Roadmap and Reports: roadmap, báo cáo và xuất PNG/Excel.
  • Administration: quản trị dự án, quản trị hệ thống, tích hợp và token.
  • Troubleshooting: lỗi đăng nhập, database, attachment, export, webhook và help file.
