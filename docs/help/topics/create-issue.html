Tạo issue mới

Tạo task, bug, story, epic hoặc subtask trong project hiện tại.

Điều kiện trước khi thao tác

  • Tài khoản đang có quyền tạo issue trong project active.
  • Project đã được chọn trên sidebar.

Thực hiện

  1. Mở màn hình Issue hoặc dùng nút tạo issue nếu đang ở toolbar tương ứng.
  2. Chọn loại issue cần tạo như Task, Bug, Story, Epic hoặc Subtask.
  3. Nhập tiêu đề, mô tả và các trường cần thiết như priority, assignee, story point, component hoặc version.
  4. Kiểm tra trường reporter. Mặc định trường này là người đang đăng nhập; chỉ thay đổi khi bạn đang tạo thay cho người khác trong cùng project.
  5. Nếu issue thuộc sprint hiện tại hoặc sprint kế tiếp, chọn sprint ở danh sách. Chỉ các sprint Planned hoặc Active mới dùng cho issue mới.
  6. Nhấn lưu để tạo issue.

Kết quả mong đợi

  • Issue mới xuất hiện trong project active và có key riêng.
  • Nếu gắn sprint, issue sẽ xuất hiện ở backlog hoặc board của sprint tương ứng tùy trạng thái hiện tại.
  • Nếu không gắn sprint, issue nằm trong backlog chung của project.
