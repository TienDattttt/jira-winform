Xuất PNG và Excel

Lưu nhanh báo cáo dưới dạng hình ảnh hoặc workbook Excel để chia sẻ và lưu trữ.

Xuất PNG

  1. Mở màn hình Báo cáo và chọn tab bạn muốn xuất.
  2. Bấm Xuất PNG.
  3. Chọn thư mục lưu file và xác nhận lưu.

Xuất Excel

  1. Mở màn hình Báo cáo và chọn project hoặc sprint cần làm báo cáo.
  2. Bấm Xuất Excel.
  3. Chọn vị trí lưu file .xlsx.
  4. Mở file bằng Excel hoặc LibreOffice để xem 4 sheet: Tổng quan, Sprint Report, Velocity và Tất cả Issues.

Lưu ý

  • Người dùng cần quyền ghi vào thư mục lưu file.
  • File Excel phù hợp cho nhu cầu lưu trữ, lọc dữ liệu và gửi quản lý; file PNG phù hợp cho trình chiếu nhanh.
