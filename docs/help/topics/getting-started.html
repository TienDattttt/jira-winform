Getting Started

Luồng bắt đầu làm việc cho người dùng cuối.

First sign-in

  1. Mở ứng dụng và nhập tài khoản, mật khẩu hoặc dùng SSO nếu môi trường có bật.
  2. Chọn project cần làm việc trong bộ chọn project ở sidebar.
  3. Kiểm tra breadcrumb để xác nhận bạn đang ở đúng project.
