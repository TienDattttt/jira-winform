User Interface

Bố cục màn hình chính và các thành phần điều hướng.

Main window layout

Area Purpose
Sidebar Điều hướng giữa các module chính của ứng dụng.
Project switcher Chọn project đang active.
Top bar Hiển thị breadcrumb, search, notification và các action theo ngữ cảnh.
Content area Hiển thị màn hình nghiệp vụ hiện tại.

Navigation rules

  • Nội dung màn hình phụ thuộc vào project đang active.
  • Một số menu chỉ xuất hiện với người dùng có quyền tương ứng.
  • Search trên thanh trên cùng áp dụng theo ngữ cảnh của màn hình đang mở.

Notifications and Help

  • Notification hiển thị các sự kiện gần đây liên quan tới issue, assignment, comment hoặc sprint.
  • Menu Help mở file trợ giúp CHM đi kèm bản cài đặt.
