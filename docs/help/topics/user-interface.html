Giao diện chính

Tổng quan về bố cục shell, mô hình điều hướng, ô tìm kiếm, thông báo và mục trợ giúp.

Các khu vực chính

Khu vực Mục đích
Sidebar Chứa project switcher, menu điều hướng, khu vực tài khoản hiện tại và thao tác đăng xuất.
Thanh trên Hiển thị breadcrumb, ô tìm kiếm của shell, thông báo và các hành động theo ngữ cảnh.
Vùng nội dung Hiển thị Dashboard, Board, Issue, Sprint, Roadmap, Reports hoặc Settings tùy theo menu đang chọn.

Nguyên tắc điều hướng

  • Breadcrumb luôn phản ánh project đang active và màn hình hiện tại.
  • Ô tìm kiếm của shell chuyển từ khóa xuống các form con có hỗ trợ search hoặc filter.
  • Menu Users chỉ hiện với system administrator.
  • Menu Help mở file CHM đi kèm thông qua HTML Help.
