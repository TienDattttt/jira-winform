Administration

Các chức năng quản trị ở mức project và mức hệ thống.

Project administration

Area Usage
Members and roles Thêm thành viên, đổi project role, xóa thành viên khỏi project.
Board and workflow Quản lý cột board, workflow transition và WIP limit.
Permissions Kiểm soát quyền xem, tạo, sửa, xóa và transition issue.
Labels, components, versions Quản lý metadata phục vụ phân loại và phát hành.

System administration

  • Quản lý tài khoản người dùng hệ thống.
  • Reset mật khẩu, khóa hoặc mở khóa tài khoản.
  • Gán system role theo nhu cầu vận hành.

Integrations and tokens

  • Cấu hình GitHub, Confluence và webhook ở mức project.
  • Theo dõi webhook delivery history.
  • Tạo và thu hồi API token theo phạm vi sử dụng.
Quản trị hệ thống không thay thế membership trong project. Người dùng vẫn phải là member của project để thao tác dữ liệu dự án đó.
