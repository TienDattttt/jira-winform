Webhook và tích hợp

Cấu hình webhook, GitHub và Confluence ở cấp project.

Webhook

  1. Mở Cài đặt của project và vào tab webhook.
  2. Thêm endpoint mới với tên, URL, danh sách sự kiện và secret nếu cần.
  3. Dùng nút Kiểm tra để gửi thử webhook.
  4. Mở Lịch sử gửi để xem HTTP status code, payload và lần gửi gần nhất.

Tích hợp GitHub hoặc Confluence

  1. Vào tab tích hợp trong phần cài đặt project.
  2. Chọn card GitHub hoặc Confluence rồi bấm cấu hình.
  3. Nhập thông tin kết nối tương ứng, ví dụ repository hoặc owner cho GitHub, space key hoặc base URL cho Confluence.
  4. Lưu cấu hình và kiểm tra trạng thái kết nối trên thẻ tích hợp.
  5. Nếu cần gỡ tích hợp, dùng thao tác Disconnect trong card tương ứng.
