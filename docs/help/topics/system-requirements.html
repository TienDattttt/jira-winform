Yêu cầu hệ thống

Các yêu cầu tối thiểu để máy trạm chạy ứng dụng ổn định.

Hạng mục Yêu cầu
Hệ điều hành Windows 10 hoặc Windows 11 64-bit.
Kết nối dữ liệu Máy trạm phải kết nối được tới SQL Server dùng chung được chỉ định trong connection string.
Quyền truy cập tệp Người dùng phải có quyền đọc và ghi vào vị trí lưu attachment khi tính năng attachment được bật.
Runtime Nếu bản build là framework-dependent thì máy phải có .NET 8 Desktop Runtime. Bản self-contained không cần runtime riêng.
Quyền ghi log và export Ứng dụng phải ghi được log và file export vào các thư mục đích được chọn.
