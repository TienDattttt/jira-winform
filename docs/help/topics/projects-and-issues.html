Projects and Issues

Các tác vụ cốt lõi trong phạm vi project và issue.

Working with projects

  • Chỉ những project mà người dùng là thành viên mới hiển thị trong bộ chọn project.
  • Mọi màn hình nghiệp vụ đều lấy dữ liệu theo project đang active.

Working with issues

Task Notes
Create issue Áp dụng cho người dùng có quyền tạo issue trong project.
Edit issue Cập nhật summary, description, priority, assignee, sprint, labels, components và versions theo quyền hiện có.
Comment and mention Dùng để trao đổi nghiệp vụ và tạo notification cho người liên quan.
Attachments and watchers Dùng để đính kèm tài liệu và theo dõi thay đổi của issue.

Issue Navigator

Màn hình Issue dùng để xem và lọc danh sách issue trong project hiện tại. Đây là nơi phù hợp để rà soát backlog, tìm issue theo trạng thái hoặc tìm các issue cần ưu tiên xử lý.
