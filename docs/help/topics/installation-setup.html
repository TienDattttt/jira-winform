Cài đặt và cấu hình

Cách cài desktop client, cấu hình kết nối dữ liệu và kiểm tra môi trường vận hành.

Trong triển khai production, desktop client và cơ sở dữ liệu là hai thành phần tách biệt. Mỗi người dùng cài WinForms client trên máy của mình, trong khi tất cả client cùng kết nối tới một SQL Server dùng chung. Nếu triển khai có attachment hoặc tích hợp, cần chuẩn bị thêm shared storage và truy cập mạng tương ứng.

Thứ tự triển khai khuyến nghị

  1. Chuẩn bị cơ sở dữ liệu SQL Server và xác nhận connection string hoạt động.
  2. Chuẩn bị vị trí lưu attachment dùng chung nếu nhiều người dùng cần truy cập cùng một tệp.
  3. Cài hoặc publish desktop client tới các máy trạm của người dùng.
  4. Cấu hình các biến môi trường JIRACLONE_ cần thiết trước lần chạy đầu tiên.
  5. Mở ứng dụng, đăng nhập và xác nhận truy cập được project, issue, attachment và báo cáo.
