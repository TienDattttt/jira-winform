Installation and Setup

Cài đặt client desktop và cấu hình môi trường chạy.

Installation package

  1. Nhận bộ cài đặt hoặc thư mục publish từ nhóm phát hành.
  2. Cài ứng dụng theo quy trình triển khai nội bộ của doanh nghiệp.
  3. Sau khi cài, xác nhận thư mục cài đặt có đủ file thực thi, cấu hình và thư mục Help.

Required configuration

Configuration Purpose
JIRACLONE_ConnectionStrings__Default Chuỗi kết nối tới SQL Server.
App:StoragePath Đường dẫn lưu attachment.
Email:* Cấu hình SMTP cho email notification.
OAuth:* Cấu hình đăng nhập SSO nếu môi trường bật OAuth.

Post-install checks

  • Ứng dụng mở được màn hình đăng nhập.
  • Kết nối database thành công.
  • File Help\JiraClone.chm tồn tại trong bản cài đặt.
  • Attachment storage truy cập được nếu môi trường có dùng file đính kèm.
Secret hoặc chuỗi kết nối production không nên được phát hành rộng rãi trong file cấu hình gốc. Chúng cần được cấp qua cơ chế cấu hình nội bộ của môi trường triển khai.
