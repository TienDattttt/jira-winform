Tổng quan sản phẩm và phạm vi tài liệu.

Product overview

Jira Clone Desktop là ứng dụng WinForms dùng để quản lý project, issue, board, sprint, roadmap, báo cáo và các tác vụ vận hành đi kèm như notification, webhook, tích hợp và API token.

Supported roles

Role Typical usage
Viewer Theo dõi tiến độ, xem issue, board, roadmap và báo cáo.
Developer Tạo và cập nhật issue, comment, attachment, watcher và trạng thái công việc.
Project Manager / Project Admin Quản lý sprint, board, workflow, permission, thành viên và cấu hình project.
System Admin Quản lý tài khoản hệ thống, cấu hình vận hành và xử lý sự cố.

System requirements

Component Requirement
Operating system Windows 10 hoặc Windows 11
Database access Kết nối tới SQL Server của môi trường vận hành
Network access Truy cập được tới attachment storage, webhook endpoint, SMTP hoặc OAuth nếu môi trường có dùng
User credentials Tài khoản hợp lệ và membership phù hợp với project cần làm việc
