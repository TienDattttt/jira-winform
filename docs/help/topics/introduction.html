Giới thiệu

Tổng quan sản phẩm, mô hình vận hành và phạm vi của tài liệu trợ giúp này.

Jira Clone Desktop là ứng dụng WinForms dùng để quản lý project, issue, board, sprint, roadmap, báo cáo, webhook, tích hợp GitHub hoặc Confluence và API token. Tài liệu này chỉ mô tả các chức năng thực sự có trong bản build hiện tại.

Mô hình vận hành

Thành phần Mô tả
Desktop client Ứng dụng WinForms chạy trên Windows, được nhiều người dùng sử dụng trên cùng một nguồn dữ liệu dùng chung.
Cơ sở dữ liệu SQL Server được cấu hình thông qua connection string của ứng dụng.
Lưu trữ attachment Tệp được ghi vào đường dẫn cấu hình qua App:StoragePath. Trong môi trường nhiều người dùng, đây nên là vị trí dùng chung.
Phân quyền Quyền truy cập được kiểm soát bởi system role và project role.

Ai nên đọc tài liệu này

  • Viewer và Developer nên tập trung vào các topic về issue, board, sprint và báo cáo.
  • Project Manager và Project Admin nên tập trung vào sprint, workflow, thành viên, quyền, webhook và tích hợp.
  • System Admin và bộ phận hỗ trợ nên tập trung vào setup, quản lý kết nối, quản trị người dùng và xử lý sự cố.

Phạm vi

  • Tài liệu này không thay thế tài liệu code, quy trình test hoặc tài liệu API.
  • Tài liệu không mô tả các thao tác không có trong ứng dụng WinForms này, kể cả khi chúng phổ biến trong Jira thương mại.
  • Mọi bước hướng dẫn đều bám theo menu và hành vi hiện tại của ứng dụng.
