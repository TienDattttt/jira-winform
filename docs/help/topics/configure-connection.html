Cấu hình kết nối và lưu trữ

Thiết lập các biến môi trường cần thiết cho truy cập database, lưu trữ attachment và các dịch vụ tùy chọn.

Biến cấu hình chính

Biến Mục đích
JIRACLONE_ConnectionStrings__Default Connection string tới SQL Server.
JIRACLONE_App__StoragePath Đường dẫn lưu attachment. Trong môi trường nhiều người dùng nên trỏ tới vị trí dùng chung.
JIRACLONE_Email__UserName and JIRACLONE_Email__Password Thông tin xác thực SMTP cho các tính năng email của ứng dụng.
JIRACLONE_ENVIRONMENT Tên môi trường để nạp thêm file appsettings.{Environment}.json nếu có.

Thực hiện

  1. Cấu hình connection string của SQL Server qua biến môi trường thay vì nhúng bí mật vào file config phân phối cho người dùng.
  2. Trỏ App:StoragePath tới shared path nếu người dùng trên nhiều máy cần truy cập cùng một attachment.
  3. Chỉ cấu hình email hoặc OAuth nếu các tính năng đó được bật trong môi trường triển khai.
  4. Khởi động lại ứng dụng sau khi đổi biến môi trường.
Không phân phối connection string production có chứa credential thật trong installer công khai hoặc file config dùng chung.
